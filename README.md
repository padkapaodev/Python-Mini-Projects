## 1. BMI Calculator with Visual Analytics (Matplotlib)

### Project Overview
This project focuses on building an **Interactive BMI (Body Mass Index) Calculator Application** using Python. 
Beyond calculating basic health metrics, it integrates data visualization capabilities to plot the user's personal BMI score 
against standard World Health Organization (WHO) health categories.

The goal was to transform numerical personal health data into an intuitive, 
visually clear chart that maps an individual's health status in real-time.

### Tech Stack
* **Language:** Python
* **Library:** Matplotlib
* **Environment:** Jupyter Notebook
